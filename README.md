# gh-cl

> Get npm package's changelog

1. You need to install [GitHub CLI](https://cli.github.com/) first
2. `gh extension install tmkx/gh-cl`
3. `gh cl vue` to view `vue`'s changelog
   - `gh cl vuejs/core` support GitHub repo also
